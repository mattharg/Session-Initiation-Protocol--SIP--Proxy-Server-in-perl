Session-Initiation-Protocol--SIP--Proxy-Server-in-perl by mattharg

Fast Prototyping Tool: A SIP proxy server for creating and testing SIP networks and working on mashups and developments to the server.

Use this SIP proxy server as your test tool for technical testing of SIP based network applications.

Install

Configure some domains within the script before running. Just run the perl code from the command-line, such as $perl SIP_proxy.pl That's it!
